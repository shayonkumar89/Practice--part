What is the DOM, and how does it represent a web page?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a document as a tree of objects, where each object corresponds to a part of the document (like elements, attributes, and text). This allows developers to programmatically interact with and manipulate the structure, style, and content of web pages.

How do you select an HTML element using JavaScript? Provide at least two methods?

document.getElementById('element-id'): Selects an element by its ID. document.querySelector('.class-name'): Selects the first element that matches a specified CSS selector (e.g., class, id, or tag).

How can you add a new HTML element to the DOM using JavaScript?

Create the element using document.createElement('tagName'). Set its attributes and content using properties like element.innerHTML or element.setAttribute('attribute', 'value'). Append the new element to the DOM using parentElement.appendChild(newElement).

What is the difference between innerHTML and textContent?

innerHTML returns or sets the HTML markup inside an element, allowing you to insert HTML tags and elements as well as text. textContent returns or sets only the text content of an element, ignoring any HTML tags. It’s useful for inserting plain text without interpreting it as HTML.
